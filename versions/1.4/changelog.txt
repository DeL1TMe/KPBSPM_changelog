Ресурспак VanillaTweaks теперь разделён на две версии: Default и Extra. Default — основная версия без значительных изменений, Ехъга — дополнительная, устанавливается по желанию. В Extra включены твики, которые могут повлиять на работу других ресурспаков или быть не нужными для большинства пользователей.
Версии модификаций были изменены до актуальных.
Библиотека ObsidianUI была удалена.
Библиотеки: Architectury, Forge Config API Port, OctoLib, Puzzles Lib были добавлены.
Моды: BadOptimizations, HeldItemInfo, RyonamicLights, ViaFabricPlus были удалены.
Моды: HeldItemTooltips, ImmersiveUI, LambDynamicLights, Resourcify, SkinShuffle были добавлены.
Переработаны ресурспаки.
Переработан список эмоций, он был дополнен (возможно наличие дубликатов).
Переработаны бинды.
Переработаны конфигурации модификаций в т.ч. отвечающие за главное меню.
Fabric Loader обновлен до актуальной версии.
Очередной фикс уведомления о доступном обновлении сборки. Теперь для этого используется статичное изображение с хостинга, а не клавиша.
